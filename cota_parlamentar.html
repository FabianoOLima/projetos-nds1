Projeto - Analise da Cota Parlamentar¶

Utilizando dados disponibilizados pela Cmara dos Deputados

Iremos analisar os dados de 2009 2017.

A atividade ser divida em duas partes:

1 parte:

Fazendo uso de grficos, tabelas e/ou mtricas estatsticas, responder as seguintes questes:

  • a. Como se comportam os gastos com cotas parlamentares ao longo do tempo? Existe alguma tendncia de aumento ou reduo desse custo? Existe sazonalidade?
  • b. Quais foram os parlamentares que mais consumiram recursos durante o mandato 2011-2014? E quais foram os que menos consumiram recursos?
  • c. Quais so as categorias de despesas mais onerosas dentre os recursos destinados s cotas parlamentares?
  • d. Qual o custo per capita de um parlamentar em cada unidade da federao? (considerando somente as despesas de cota parlamentar)

2 parte:

Essa parte livre, ficando a cargo do candidato realizar anlises e responder suas prprias questes. Neste caso, responderei as seguintes questes:

  • a. Qual partido mais consumiu recursos ?
  • b. Qual candidato gastou mais em combustvel no perodo x ?

Etapas do projeto:¶

  • Leitura dos arquivos
  • Analise superficial dos dados
  • Limpeza do DataFrame
  • Analise e resposta das questes levantadas
  • Concluso
  • Limitaes

# Dados para descrio
def descreve(filtro):
    """
    Funo para descrever os dados apresentados no grfico
    Paramentros:
    filtro - variavel que possui filtro que ser aplicado ao grfico
    retorna:
    Texto descrevendo as informaes apresentadas no grfico    
    """
    valor_min = df_cotas[filtro].groupby('numAno')['vlrLiquido'].sum().min()
    vmin_ano = df_cotas[filtro].groupby('numAno')['vlrLiquido'].sum().idxmin()
    valor_max = df_cotas[filtro].groupby('numAno')['vlrLiquido'].sum().max()
    vmax_ano = df_cotas[filtro].groupby('numAno')['vlrLiquido'].sum().idxmax()
    valor_varia = ((valor_max - valor_min) / valor_min) * 100
    print('Teve uma variao de {:.2f}% entre os anos de {} e {}, partindo de um gasto de R$ {:.2f} para R$ {:.2f}'
         .format(valor_varia,vmin_ano,vmax_ano,valor_min,valor_max))

  • vlrLiquido : convertemos para float
